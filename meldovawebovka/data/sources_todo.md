# Zdroje pro aktualizaci webu Martina Melichara

Tento soubor slouží správci webu k ověřování faktů. AI model vložil do webu zástupná data, aby se vyhnul halucinacím.

## 1. Kde vzít statistiky (pro soubor stats.json)
Otevři `data/stats.json` a uprav hodnoty podle těchto zdrojů:

* **Transfermarkt:** [Profil Martina Melichara](https://www.transfermarkt.com/martin-melichar/profil/spieler/568832)
    * Hledej sekci "Performance data".
    * Zkontroluj aktuální počet zápasů, čistých kont a minut.
* **Livesport:** Vyhledej "FK Viagem Příbram", rozklikni záložku "Soupiska" -> Brankáři.
* **Oficiální web klubu:** [fkviagem.cz](https://www.fkviagem.cz/)
